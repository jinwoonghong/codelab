# Codelab 🚀

다양한 MVP 서비스를 빠르게 프로토타이핑하고 실험하는 공간입니다.

## 프로젝트 목록

현재 진행 중인 프로젝트들:

| # | 프로젝트명 | 설명 | 기술 스택 | 상태 |
|---|-----------|------|----------|------|
| 1 | tower-stacker | 타워스테커 모바일 웹 게임 | HTML5, JavaScript | 🚧 기획 중 |
| 2 | link-keeper | 모바일 링크 관리 PWA | HTML5, JavaScript, IndexedDB | 📝 기획 완료 |

## 폴더 구조

```
codelab/
├── projects/          # 각 MVP 프로젝트들
├── shared/           # 공통으로 사용하는 코드
├── templates/        # 프로젝트 시작 템플릿
└── docs/            # 문서 및 가이드
```

## 새 프로젝트 시작하기

1. `projects/` 폴더에 새 프로젝트 디렉토리 생성
2. 템플릿을 복사하거나 새로 시작
3. 프로젝트 README.md 작성
4. 이 파일의 프로젝트 목록 업데이트

## 사용 도구

- **Claude Code**: AI 기반 코딩 어시스턴트
- **Git/GitHub**: 버전 관리
